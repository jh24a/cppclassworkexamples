# Extra credit assignment - reverse an integer

Writing a program to print the reverse of a positive integer, and answer the question at the end of this document.

You can assume the user always input a positive integer that can be stored in an unsigned int variable, so for this assignment you do not need to consider invalid inputs.

DO NOT use existing libraries (e.g., atoi) other than the standard C/C++ operations.

For example, 

The input is

    12345

The output your program should provide is

    54321

The input is

    1200

The output is

    21

For the second example, extra 0s are removed, i.e., 0021 is printed as 21, as leading zeros are useless.

Question: If you need to store the reversed value in a variable and return it, can you simply use another unsigned int variable for the result? If not, what is the problem?
