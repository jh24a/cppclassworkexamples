# Assignment 3. Finding the largest number in an array

Writing a program to find the largest number in an array that is provided by the user. You only need to consider integers. You program can work as follows:

- Ask the user to input a value n, which decides the size of the array. Then you can allocate an array of size n.
- Ask the user to input n values, and save them in an array you generated in the previous step.
- Find out the largest value and print out.
