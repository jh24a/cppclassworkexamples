# Play with math functions

Given three floating-point numbers x, y, and z, output x to the power of y, x to the power of (y to the power of z), the absolute value of x, and the square root of (xy to the power of z). 

For example, if the input is:

5.0 6.5 3.2

the output is:

34938.6 1.29951e+279 5 262.43 


*hint:* 
- You need to include the math library in order to call these math functions
- If you do not know the name or syntax of the math functions, please use google.

# Play with string functions

Given a string s, divide it into two parts from the middle, and insert the new string "+++" there.

For example, if the input is:

"abcd"

the output is:
"ab+++cd"

*hint:*
- Let's assume the input string can always evenly divided into two parts, i.e., the length of the string is an even number.
- Try to have at least two versions of programs for the same function. For example, one version can operate on the original input, another version can generate a new string for the result.


# Finding the number

Given three integers x, y, z, find and output the largest one.

For example, if the input is 

3, 4, 2

the output is 4
