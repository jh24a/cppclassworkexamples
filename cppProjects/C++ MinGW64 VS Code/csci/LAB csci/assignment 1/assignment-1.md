# Assignment 1.

Given a string s, convert it to an integer n. We only consider the decimal system.

For example, if the input is "345", the output should be 345.

*Hints:*

- The string may include invalid characters, e.g. "1j2". In this case, return 0.
- You can use C++ standard string library. Take a look of http://www.cplusplus.com/reference/string/string/ to learn how to play with strings.
- The number can be negative, i.e., the string is "-10".
